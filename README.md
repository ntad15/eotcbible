# eotcbible
Interactive bible for EOTC

# Steps before you get started
1. install yarn at https://classic.yarnpkg.com/lang/en/docs/install/#mac-stable
2. do this in ./eotcbible directory - ''yarn global add expo-cli''
# Get started
1. Clone Repo
2. cd ./frontend
3. do ''yarn install''
4. 'expo start'
